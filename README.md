# 📌 Commit Convention

| 깃모지 | 사용예시                 |
| ------ | :----------------------- |
| 🎉     | 프로젝트 시작            |
| ✨     | 새 기능                  |
| 🎨     | 코드 구조/형태 개선      |
| ⚡️    | 성능 개선                |
| 💄     | UI/스타일 파일 추가/수정 |
| ♻️     | 코드 리팩토링            |
| 🐛     | 버그 수정                |
| 🚑     | 긴급 수정                |
| 🔒     | 보안 이슈 수정           |
| 💡     | 주석 추가/수정           |
| 🔥     | 코드/파일 삭제           |
